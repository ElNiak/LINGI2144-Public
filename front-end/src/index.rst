Welcome to LINGI2144 Secured System Engineering!
=================================================

.. only:: html

The objective of the course is to give an introduction to software security. We will first discuss the concepts of security and software attack. We will then analyze software vulnerabilities and we will study protections. Finally, an introduction to malware analysis will be presented.   - Introduction to cyber security and virology
   - Threat Vulnerabilities and Attack
   - Vulnerability of C language (Buffer overflow, heap, format string) and protections
   - Shell code: write in C / assemble
   - Malware classification
   - Malware detection
   - Static, dynamic and artificial intelligence techniques.
